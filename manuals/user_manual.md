# 利用者向け手順書

## 1. 使い方

1. アプリを起動する。
2. 入力方式を選ぶ。
3. Markdownを入力、またはファイルを指定する。
4. 出力先、フォント名、フォントサイズを設定する。
5. `変換実行` を押す。
6. 完了表示後に出力ファイルを確認する。

## 2. エラー時の基本対応

- 入力が空でないか確認する。
- 入力ファイルの場所が正しいか確認する。
- 出力先フォルダに保存権限があるか確認する。
- 必要に応じて運用担当部署へ連絡する。

## 3. 注意事項

- 変換後の文書内容は必ず目視確認する。
- 個人情報を含む文書は組織ルールに従って管理する。
