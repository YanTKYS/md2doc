# 管理者向け手順書

## 1. 配置手順

- 配布先はWindows端末とする。
- Office Standard導入済み端末へ配置する。

## 2. 更新手順

1. 新版を取得する。
2. 旧版を退避する。
3. 新版に差し替える。
4. 起動・基本変換を確認する。

## 3. ロールバック手順

- 問題発生時は旧版へ戻す。
- 版番号と変更点を記録する。

## 4. 障害対応（一次切り分け）

- Word起動不可時はOffice状態を確認する。
- COM関連エラー時は再起動後に再実行する。

## 5. 権限管理

- 出力先フォルダに書き込み権限が必要。
- 共有端末では保存先ルールを統一する。
