# 運用担当部署向け手順書

## 1. 一次対応

- エラーメッセージを確認する。
- 入力ファイル存在、出力先権限、Office起動可否を確認する。
- 再実行で再現するか確認する。

## 2. 業務判断

- 変換後文書の内容妥当性は利用部署で確認する。
- 体裁差異が業務上許容可能か判断する。

## 3. DX担当への相談境界

- アプリが起動しない。
- 毎回同一箇所で変換失敗する。
- Office連携エラーが継続する。

## 4. 記録して引き継ぐ情報

- 発生日時
- 端末情報（OS、Office版、ビット数）
- 実行した入力ファイル種別（本文は記録しない）
- 表示されたエラーメッセージ
