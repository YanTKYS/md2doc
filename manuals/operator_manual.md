# 運用担当部署向け手順書

## 1. システム概要

md2doc は Markdown → Word（`.docx`）変換ツールです。  
**Open XML 方式**（標準）では Microsoft Word を起動せずに変換します。  
Word COM 方式（互換確認用）は Microsoft Word が必要です。

## 2. 一次対応

1. エラーメッセージを確認する。
2. 以下を順に確認する。
   - 入力ファイルが存在するか（ファイル入力の場合）
   - 出力先フォルダへの書き込み権限があるか
   - Word COM 方式使用時は、Microsoft Word が起動できるか確認する
3. 変換エンジンを **Open XML 方式** に切り替えて再試行する。
4. 再実行で再現するか確認する。

## 3. 業務判断

- 変換後文書の内容妥当性は利用部署で確認する。
- 体裁差異（フォント・行間・段落間隔等）が業務上許容可能か判断する。
- Open XML 方式と Word COM 方式で出力差異がある場合は Open XML 方式の出力を優先する。

## 4. DX 担当への相談境界

以下の場合は DX 担当へ相談する。

- アプリが起動しない。
- 毎回同一箇所で変換失敗する。
- Open XML 方式・Word COM 方式の両方で変換失敗する。
- 出力 docx が Word で開けない。

## 5. 記録して引き継ぐ情報

- 発生日時
- 端末情報（OS バージョン、.NET バージョン）
- Word COM 方式使用時: Office バージョン・ビット数
- 変換エンジン（Open XML 方式 / Word COM 方式）
- 表示されたエラーメッセージ
- 入力ファイル種別（本文は記録しない）
